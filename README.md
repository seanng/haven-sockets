# Express Starter Kit

This is a customer starter kit for Express.

## Getting Started

1. Clone this repo
2. `cd` into this repo 
3. You might want to rename the `package.json`
4. `yarn && yarn dev`

If you plan to `yarn dev:watch`, you would need (nodemon)[https://github.com/remy/nodemon].
